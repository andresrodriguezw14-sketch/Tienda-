# Tienda Yulibeth 🏪
Sistema de inventario desarrollado en Python desde Simacota, Santander

## Funcionalidades
- CRUD completo: Agregar, vender, consultar y reducir stock
- Programacion Orientada a Objetos: Clases, metodos y atributos
- Validaciones de stock y productos inexistentes
- Sistema de descuentos aleatorios con libreria random
- Menu interactivo con match-case de Python 3.10
- Limpieza de pantalla compatible con Windows y Linux
